# Concerten

Een iOS-app voor het loggen van concerten en festivals waar je bent geweest, met cijfers voor optreden en geluid, foto's, notities, en — later — gedeelde overzichten met vrienden.

## Tech stack

- **SwiftUI** voor iOS 17+
- **Supabase** (Postgres + Auth + Storage)
- **Setlist.fm** + **Spotify** als externe databronnen
- **Git + GitHub** voor versiebeheer
